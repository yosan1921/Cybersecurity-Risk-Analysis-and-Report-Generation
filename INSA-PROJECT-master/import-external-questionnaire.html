📥 Import External Questionnaire

Import questionnaires from external systems or APIs

ℹ️ Paste Your JSON Data

Copy and paste questionnaire data from your external system in JSON format.

ℹ️ Upload JSON File

Upload a JSON file containing questionnaire data from your external system.

ℹ️ Fetch from API

Provide an API endpoint URL that returns questionnaire data in JSON format.

📚 Supported Data Formats

The import system automatically detects and handles multiple JSON formats:

Format 1: Template with Sections

{
  "questionnaireName": "Assessment",
  "sections": [{
    "sectionTitle": "Security",
    "questions": [{
      "questionText": "Question?",
      "answer": "Yes"
    }]
  }]
}

Format 2: Flat Array

[{
  "question": "Question?",
  "answer": "Yes",
  "section": "Security"
}]

Format 3: Questions Object

{
  "title": "Assessment",
  "questions": [{
    "question": "Question?",
    "answer": "Yes",
    "section": "Security"
  }]
}

Format 4: Custom External

{
  "items": [{
    "text": "Question?",
    "response": "Yes",
    "category": "Security"
  }]
}

🔑 Field Mapping

The system automatically maps various field names:

Question: questionText, question, text, description
Answer: answer, response
Section: section, category, domain, group
Level: level, priority (defaults to 'operational')
