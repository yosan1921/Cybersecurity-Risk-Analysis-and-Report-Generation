📥 Import External Questionnaire

Import questionnaires from other systems with or without answers

ℹ️ Supported Formats

  • With Answers: Import completed questionnaires from external systems
  • Without Answers: Import question templates to be filled out later
  • Format: JSON with sections and questions

📖 JSON Format Example

Use this format for importing questionnaires:

{
  "title": "Security Assessment Q1 2024",
  "company": "Acme Corporation",
  "filledBy": "John Doe",
  "role": "IT Security Manager",
  "filledDate": "2024-03-07",
  "sections": [
    {
      "sectionTitle": "Physical Security Controls",
      "questions": [
        {
          "questionText": "Are all data center entry points protected?",
          "answer": "Yes"
        },
        {
          "questionText": "Is CCTV installed in critical areas?",
          "answer": "Partially Implemented"
        }
      ]
    },
    {
      "sectionTitle": "Network Security",
      "questions": [
        {
          "questionText": "Is the network properly segmented?",
          "answer": "No"
        }
      ]
    }
  ]
}

💡 Tips

  • The answer field is optional - omit it for templates
  • Valid answers: "Yes", "No", "Partially Implemented", "Not Applicable"
  • You can add a level field: "operational", "tactical", or "strategic"
